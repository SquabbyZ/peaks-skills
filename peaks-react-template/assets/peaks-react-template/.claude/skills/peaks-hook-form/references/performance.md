# Performance Optimization

## Component Memoization

All form components generated by peaks-hook-form are optimized with React's `memo()` to prevent unnecessary re-renders.

### Form Component Template

When you create a form template using `create_form_template.py`, the generated component is automatically wrapped with `memo()`:

```tsx
import { memo } from 'react';
import { FormProvider, useForm, useWatch } from 'react-hook-form';

function DatasetSettingsAntd() {
  // Component logic...
}

export default memo(DatasetSettingsAntd);
```

### Benefits of memo()

- **Prevents Unnecessary Re-renders**: The component only re-renders when its props change
- **Improves Performance**: Reduces rendering overhead in large forms
- **Optimized for React Hook Form**: Works seamlessly with `FormProvider` context

### Form Field Components

All form field components (Hook\*FormItem) are also wrapped with `memo()`:

```tsx
import { memo } from 'react';

export const HookNameFormItem = memo(
  ({
    prop,
    label,
    tooltip,
    placeholder,
    required,
    ...restInputProps
  }: NameFormItemProps) => {
    const { control, trigger, formState } = useFormContext();

    return (
      <Controller
        name={prop}
        control={control}
        render={({ field }) => {
          // Component implementation...
        }}
      />
    );
  },
);
```

## Best Practices

### 1. Use useWatch Strategically

Only watch the fields you actually need:

```tsx
// ✅ Good - Watch only necessary fields
const name = useWatch({
  control: form.control,
  name: 'name',
});

// ❌ Avoid - Watching entire form
const allValues = useWatch({
  control: form.control,
});
```

### 2. Leverage defaultValues

Always use the generated `defaultValue` for optimal performance:

```tsx
const form = useForm<z.infer<typeof RagConfigSchema>>({
  resolver: zodResolver(RagConfigSchema),
  defaultValues: defaultValue, // ✅ Uses optimized default values
});
```

### 3. Memoize Submit Handlers

For complex submit logic, consider wrapping the handler with `useCallback`:

```tsx
import { useCallback } from 'react';

const onSubmit = useCallback(async (data: z.infer<typeof RagConfigSchema>) => {
  // Complex logic...
}, []);
```

### 4. Optimize Field Components

When adding custom logic to field components:

- Use `memo()` for components with complex rendering
- Use `useCallback` for event handlers
- Avoid inline function definitions in JSX
